Instrucciones

Esta aplicación te permite realizar auditorías de cumplimiento de PPR según la norma TS FSSC 22002.

  • Selecciona "Cumple", "Cumple Parcial", "No Cumple" o "No Aplica" para cada requisito
  • Para los casos de "Cumple Parcial" o "No Cumple" debes adjuntar evidencia fotográfica
  • Puedes agregar observaciones para cada requisito
  • Al finalizar, se generará un reporte ejecutivo en PDF
  • Puedes compartir el reporte por correo o WhatsApp
  • Los administradores recibirán notificaciones de incumplimientos

Toda la información se almacena de forma segura en Google Sheets para su posterior análisis.
